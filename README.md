# 海淘志（HTZ）- 简历项目版

海淘志是一个面向本地生活场景的高并发交易系统项目，适合在简历与面试中展示后端工程化能力、分布式系统设计能力与性能稳定性治理能力。

## 技术栈

### 后端核心
- Java 17+
- Spring Boot 3.x
- Spring MVC
- MyBatis-Plus
- Maven 多模块架构

### 数据与中间件
- MySQL（核心业务数据存储）
- Redis（缓存、限流、签到统计、排行榜等）
- Redisson（分布式锁、延迟任务能力）
- Kafka（异步解耦、削峰填谷、可靠消费）
- ShardingSphere（分库分表与路由治理）

### 前端与交互
- Vue 3
- Pinia
- Vite

### 工程能力关键词
- Lua 原子脚本
- 分布式限流与令牌机制
- 幂等控制与失败重试
- 最终一致性与对账补偿

## 功能

### 1. 用户与认证
- 实现手机验证码登录与会话管理，支撑分布式场景下的登录态共享。
- 完成用户资料与账号基础能力，形成可扩展的用户域模型。

### 2. 商户查询与内容互动
- 提供商户列表、详情与类型检索能力，优化高频查询链路。
- 支持探店内容发布与互动行为，覆盖本地生活内容化场景。

### 3. 优惠券秒杀核心链路
- 构建秒杀资格校验、库存扣减、订单创建的完整交易闭环。
- 通过异步消息化处理提升高并发下的系统吞吐与稳定性。

### 4. 社交关系与用户活跃
- 实现关注/取关、共同关注、点赞等社交能力。
- 沉淀可复用的关系链与互动数据模型。

### 5. 运营增长与数据统计
- 支持用户签到、UV 统计、店铺榜单等运营能力。
- 为活动运营提供数据化支撑与效果观测基础。

### 6. 稳定性与工程化保障
- 系统化治理缓存穿透、击穿、雪崩等典型缓存问题。
- 通过分布式锁保障并发写安全，避免热点冲突。
- 构建消息可靠投递、失败重试与补偿机制，提升链路可靠性。
- 结合分库分表与全局 ID 设计，支撑业务规模扩展。
